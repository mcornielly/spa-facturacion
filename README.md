### Installation
---

`$ git clone https://github.com/mcornielly/spa-invoice.git`

`$ cd spa-invoice`

`$ composer install`

`$ npm install`

First steps:
------------
### Remember
Create and configure the .env file and configure the following
`
 To create the root/.env file
`

Generate key:
------------
`
$ spa-invoice /php artisan key:generate
`
Execute the command:
----
`
$ php artisan migrate
`
Run the server:
`
$php artisan serve
`
