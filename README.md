# Single Page Aplication with Laravel 5.7 and Vue.js 2.6.x

### Framework and Libraries
- Laravel 5.7
- Vue.js 5.6.x
- Vue Router
- axios

### Provides
- CRUD
- Single Page Aplication 
- Pagination
- Progressbar
- Forms with selectable product and customers
